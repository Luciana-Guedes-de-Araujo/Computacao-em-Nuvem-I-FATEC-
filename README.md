# Computacao-em-Nuvem-I-FATEC-
Neste repositório está disponível resumos de aulas teóricas e de laboratório da matéria de computação em nuvem do curso DSM.


18/11 - Aula

Início do conteúdo da disciplina.

Introdução à nuvem.

Explicação dos papéis de cliente, API e servidor em um sistema.

Introdução ao conceito de computação em nuvem e ao modelo de pagamento conforme o uso.

Tipos de ambientes de trabalhos: local, em nuvem e híbrido.
*****************************************************************************************

25/11 Aula

Revisão dos tópicos estudados anteriormente.

Estudo sobre máquinas virtuais e o funcionamento das zonas de disponibilidade.

Primeiros conceitos sobre SaaS (Software como Serviço).
*****************************************************************************************

29/11 – Aula de Reposição

Discussão sobre o modelo de responsabilidade compartilhada na nuvem.

Configuração inicial da conta no Azure e exploração dos recursos.

Pontos importantes ao montar uma máquina virtual.

Recomendação de estudo: slides até o número 10 para a P1.
*****************************************************************************************

02/12 - Aula

Definição das ferramentas a serem usadas antes da criação de uma máquina virtual.

Definição de contêineres, Docker, Imagem, Marcas.

Certificação AZ-900.

Tipos de serviços: IaaS, PaaS e SaaS.
*****************************************************************************************

06/12 – Aula de Reposição

Conceitos sobre os campos da plataforma Azure para criação de máquina virtual.

DMZ, zonas soberanas (Governamental).

Atividade prática focada na criação de uma máquina virtual.
*****************************************************************************************

09/12 - Aula

Revisão dos conceitos principais do Azure.

Atividade para treino de criação de máquinas e virtuais e pings. 
*****************************************************************************************

12/12 - Aula de Reposição

Contêineres: instâncias, serviços, kubernetes, funções.

Redes VNet.

Dimensionamento de máquinas virtuais.
